INTRODUCTION

Voice of the People is a project dedicated to reinvigorating the news economy. This project seeks to honor core tenets of journalism -- the dedication to truth seeking, accountability and giving a voice to those who need it most -- by shifting the value of the news product from the website adn features itself to the free marketplace that benefits users and producers alike.
